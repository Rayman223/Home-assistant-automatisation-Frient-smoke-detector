# List of all helpers

- **input_boolean.alarm_is_active**  
  Indicates the current state of the alarm.

- **input_boolean.smoke_alarm_silence**  
  Enables silence mode (stops the alarm).

- **switch.groupe_alarme**  
  Contains all alarm actuators of every smoke detector.

- **binary_sensor.groupe_detection_de_fumee**  
  Contains all smoke sensors of every smoke detector.

- **input_text.smoke_alarm_source**  
  Name of the smoke detector that is currently detecting smoke.

- **timer.smoke_source_block**  
  Duration during which the alarm is blocked.
